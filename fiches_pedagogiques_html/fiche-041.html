Compétence : Utiliser l’attribut target dans un lien

Date : 20/06/2025

Version : v1.0

Objectif

Comprendre l'utilisation de l'attribut target dans un lien HTML.

Théorie

L'attribut target dans un lien HTML permet de spécifier où le document lié doit s'ouvrir. Les valeurs courantes de l'attribut target incluent :

  • _self : Ouvre le lien dans la même fenêtre ou onglet (valeur par défaut).
  • _blank : Ouvre le lien dans une nouvelle fenêtre ou onglet.
  • _parent : Ouvre le lien dans la fenêtre parent.
  • _top : Ouvre le lien dans la fenêtre la plus haute (utile pour les frames).

Exemple

Voici un exemple de code HTML utilisant l'attribut target :

<!DOCTYPE html>
<html lang="fr">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Exemple target</title>
</head>
<body>
    <a href="https://www.example.com" target="_blank">Ouvrir dans un nouvel onglet</a>
</body>
</html>
