Compétence : Utiliser des éléments imbriqués

Date : 21/06/2025

Version : v1.0

Objectif

Comprendre comment utiliser des éléments HTML imbriqués pour structurer le contenu de manière sémantique et logique.

Théorie

Les éléments HTML imbriqués permettent de structurer le contenu de manière hiérarchique et sémantique. L'imbrication consiste à placer un élément HTML à l'intérieur d'un autre élément HTML. Cela permet de créer des relations parent-enfant entre les éléments, ce qui est essentiel pour une bonne structuration du document.

Voici quelques exemples d'éléments imbriqués :

  • Les listes imbriquées : une liste à puces (<ul>) contenant des éléments de liste (<li>), qui eux-mêmes peuvent contenir des listes imbriquées.
  • Les sections imbriquées : une section (<section>) contenant des articles (<article>), qui eux-mêmes peuvent contenir des sections.
  • Les éléments de formulaire imbriqués : un formulaire (<form>) contenant des groupes de champs (<fieldset>) et des légendes (<legend>).

Exemple

Voici un exemple de code HTML utilisant des éléments imbriqués :


<!DOCTYPE html>
<html lang="fr">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Exemple d'éléments imbriqués</title>
</head>
<body>
    <section>
        <h1>Titre de la section</h1>
        <article>
            <h2>Titre de l'article</h2>
            <p>Paragraphe de l'article.</p>
            <ul>
                <li>Élément de liste</li>
                <li>Élément de liste imbriqué
                    <ul>
                        <li>Sous-élément de liste</li>
                    </ul>
                </li>
            </ul>
        </article>
    </section>
</body>
</html>
