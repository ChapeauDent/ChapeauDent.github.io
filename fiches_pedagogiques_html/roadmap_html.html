Roadmap d'apprentissage HTML

🔰 Niveau Initial – Découverte

  • Comprendre l’usage principal du HTML
  • Utiliser des éléments HTML simples (<p>, <h1>, <ul>, etc.)
  • Ajouter des attributs simples (src, alt, href, etc.)
  • Créer une structure de document (<!DOCTYPE html>, <html>, <head>, <body>)
  • Ajouter un titre de section (<h1> à <h6>)
  • Afficher une liste (<ul>, <ol>, <li>)
  • Créer un tableau simple (<table>, <tr>, <td>)
  • Ajouter une image (<img>)
  • Utiliser un lien (<a>)
  • Ajouter un saut de ligne (<br>)
  • Créer une structure de page simple
  • Utiliser un conteneur de bloc (<div>)
  • Utiliser un formulaire de base (<form>)

🧱 Niveau Basique – Mise en pratique

  • Utiliser des éléments imbriqués
  • Ajouter des attributs simples (id, class, title)
  • Écrire un commentaire (<!-- commentaire -->)
  • Ajouter un titre à la page (<title>)
  • Ajouter un paragraphe (<p>)
  • Ajouter un titre à une image (alt, title)
  • Utiliser l’attribut href dans un lien
  • Créer un conteneur en ligne (<span>)
  • Utiliser des éléments de mise en forme (<strong>, <em>, <mark>)
  • Ajouter un formulaire avec champs (<input>, <textarea>, <select>)
  • Soumettre un formulaire (<button type="submit">)
  • Utiliser les entités HTML (&nbsp;, &lt;, &copy;, etc.)
  • Créer des liens vers des ancres internes (<a href="#section1">)
  • Utiliser les balises sémantiques de base (<main>, <header>, <footer>, <nav>)

⚙️ Niveau Opérationnel – Maîtrise des structures

  • Identifier les structures HTML (hiérarchie, indentation)
  • Utiliser des éléments vides (<br>, <hr>, <img>)
  • Définir le doctype (<!DOCTYPE html>)
  • Identifier les niveaux de titres et leur usage sémantique
  • Utiliser des attributs complexes dans un tableau (colspan, rowspan)
  • Ajouter une vidéo (<video>)
  • Ajouter des intitulés aux champs de formulaire (<label>)
  • Structurer une page avec des balises sémantiques HTML5
  • Ajouter des métadonnées dans le <head> (<meta charset="UTF-8">, description, viewport)
  • Utiliser les balises <fieldset> et <legend> dans les formulaires
  • Gérer les erreurs de saisie avec les attributs HTML (required, pattern, etc.)

🧠 Niveau Avancé – Sémantique et accessibilité

  • Différencier les éléments de bloc et en ligne
  • Utiliser des attributs booléens (checked, disabled, required)
  • Utiliser l’attribut target dans un lien
  • Regrouper les cellules d’un tableau (<thead>, <tbody>, <tfoot>)
  • Choisir l’élément pertinent selon le contenu (<article>, <section>, <aside>, etc.)
  • Choisir le type d’entrée pertinent (type="email", type="date", etc.)
  • Choisir le bon formulaire selon le contexte
  • Accessibilité (a11y) : utiliser les attributs aria-*, les rôles (role="button", etc.)
  • Utiliser les balises <details> et <summary> pour créer des contenus repliables
  • Créer des tableaux accessibles (avec <caption>, <th scope="col">, etc.)
  • Utiliser les balises <time>, <mark>, <abbr>, <cite> pour enrichir le contenu sémantique

🧬 Niveau Expert – Optimisation et personnalisation

  • Ajouter des attributs complexes (data-*)
  • Afficher une liste ordonnée (<ol>)
  • Ajouter une image vectorielle (<svg>)
  • Utiliser les balises <template> et <slot> (préparation à l’usage de Web Components)
  • Intégrer HTML avec JavaScript pour manipuler le DOM
  • Optimiser le SEO avec des balises structurées (<meta>, balises Open Graph, etc.)
  • Préparer une page pour le responsive design (viewport, structure fluide)
