Compétence : Ajouter des métadonnées dans le <head> (<meta charset="UTF-8">, description, viewport)

Date : 20/06/2025

Version : v1.0

Objectif

Comprendre comment ajouter des métadonnées dans la balise <head> en HTML.

Théorie

Les métadonnées dans la balise <head> d'un document HTML fournissent des informations supplémentaires sur la page web. Elles ne sont pas affichées directement sur la page, mais elles sont utilisées par les navigateurs, les moteurs de recherche et d'autres services web pour comprendre et traiter le contenu de la page.

Voici quelques exemples de métadonnées couramment utilisées :

  • <meta charset="UTF-8"> : Spécifie l'encodage des caractères utilisé par la page. UTF-8 est l'encodage le plus courant et recommandé.
  • <meta name="description" content="Description de la page"> : Fournit une description de la page qui peut être utilisée par les moteurs de recherche pour afficher un extrait dans les résultats de recherche.
  • <meta name="viewport" content="width=device-width, initial-scale=1.0"> : Contrôle la mise en page et le dimensionnement de la page sur les appareils mobiles.

Exemple

<!DOCTYPE html>
<html lang="fr">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <meta name="description" content="Exemple de métadonnées dans le head en HTML">
    <title>Exemple de Métadonnées</title>
</head>
<body>
    <h1>Bienvenue sur notre site web</h1>
</body>
</html>
