Compétence : Créer un élément conteneur de bloc

Date : 21/06/2025

Version : v1.0

Objectif

Apprendre à créer et utiliser un élément conteneur de bloc en HTML.

Théorie

En HTML, un élément conteneur de bloc est utilisé pour regrouper d'autres éléments HTML afin de les organiser et de les styliser. Les éléments conteneurs de bloc les plus courants sont les balises <div> et <section>.

Les éléments de bloc occupent toute la largeur disponible de leur conteneur parent et commencent toujours sur une nouvelle ligne. Ils sont souvent utilisés pour structurer le contenu d'une page web de manière logique et lisible.

Voici quelques utilisations courantes des éléments conteneurs de bloc :

  • Regrouper des éléments pour appliquer des styles CSS communs.
  • Créer des sections distinctes dans une page web.
  • Faciliter la manipulation du DOM avec JavaScript.

Exemple

Voici un exemple simple de code HTML utilisant un élément conteneur de bloc :


<!DOCTYPE html>
<html lang="fr">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Exemple de conteneur de bloc</title>
</head>
<body>
    <div style="border: 1px solid #000; padding: 10px;">
        <h2>Titre de la section</h2>
        <p>Ceci est un paragraphe à l'intérieur d'un conteneur de bloc.</p>
        <p>Un autre paragraphe dans le même conteneur.</p>
    </div>
</body>
</html>
