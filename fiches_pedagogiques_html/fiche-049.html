Compétence : Utiliser les balises <time>, <mark>, <abbr>, <cite> pour enrichir le contenu sémantique

Date : 20/06/2025

Version : v1.0

Objectif

Comprendre l'utilisation des balises <time>, <mark>, <abbr>, <cite> pour enrichir le contenu sémantique en HTML.

Théorie

Les balises HTML <time>, <mark>, <abbr>, <cite> permettent d'enrichir le contenu sémantique d'une page web :

  • <time> : Utilisée pour représenter des dates et des heures.
  • <mark> : Utilisée pour marquer du texte qui a une importance particulière ou qui doit être mis en évidence.
  • <abbr> : Utilisée pour indiquer une abréviation ou un acronyme, avec l'attribut title pour fournir la forme complète.
  • <cite> : Utilisée pour citer une source ou une référence.

Exemple

Voici un exemple d'utilisation de ces balises :

<!DOCTYPE html>
<html lang="fr">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Exemple de balises sémantiques</title>
</head>
<body>
    <p>L'événement aura lieu le <time datetime="2025-06-20">20 juin 2025</time>.</p>
    <p>Le terme <abbr title="HyperText Markup Language">HTML</abbr> signifie HyperText Markup Language.</p>
    <p>Pour plus d'informations, consultez le livre <cite>HTML & CSS: Design and Build Websites</cite>.</p>
    <p>Ce texte est <mark>très important</mark>.</p>
</body>
</html>
