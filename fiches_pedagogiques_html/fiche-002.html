Compétence : Utiliser des éléments HTML simples

Date : 21/06/2025

Version : v1.0

Objectif

Apprendre à utiliser les éléments HTML simples pour structurer une page web.

Théorie

Les éléments HTML simples sont les blocs de construction de base pour créer une page web. Voici quelques-uns des éléments les plus couramment utilisés :

  • <p> : Utilisé pour définir un paragraphe.
  • <h1> à <h6> : Utilisé pour définir les titres de section, du plus important (<h1>) au moins important (<h6>).
  • <ul> et <ol> : Utilisés pour créer des listes non ordonnées (<ul>) et ordonnées (<ol>).
  • <li> : Utilisé pour définir un élément de liste.
  • <table>, <tr>, <td> : Utilisés pour créer des tableaux.
  • <img> : Utilisé pour ajouter des images.
  • <a> : Utilisé pour créer des liens.
  • <br> : Utilisé pour ajouter un saut de ligne.
  • <div> : Utilisé pour créer des conteneurs de bloc.
  • <form> : Utilisé pour créer des formulaires.

Exemple

Voici un exemple simple de code HTML utilisant ces éléments :


<!DOCTYPE html>
<html lang="fr">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Exemple HTML</title>
</head>
<body>
    <h1>Titre de la page</h1>
    <p>Ceci est un paragraphe.</p>
    <ul>
        <li>Élément de liste 1</li>
        <li>Élément de liste 2</li>
    </ul>
    <table>
        <tr>
            <td>Cellule 1</td>
            <td>Cellule 2</td>
        </tr>
    </table>
    <img src="image.jpg" alt="Description de l'image">
    <a href="https://example.com">Lien vers un site</a>
    <br>
    <div>Conteneur de bloc</div>
    <form>
        <input type="text" name="nom" placeholder="Votre nom">
        <button type="submit">Envoyer</button>
    </form>
</body>
</html>
