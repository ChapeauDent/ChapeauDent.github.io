Compétence : Expliquer l’usage principal du HTML

Date : 21/06/2025

Version : v1.0

Objectif

Comprendre l'usage principal du HTML dans le développement web.

Théorie

HTML (HyperText Markup Language) est le langage standard utilisé pour créer des pages web. Il permet de structurer le contenu et de définir la sémantique des éléments présents sur une page. HTML est composé de balises qui entourent le contenu pour indiquer au navigateur comment afficher ce contenu.

Les principales utilisations de HTML incluent :

  • La création de la structure de base d'une page web avec des éléments tels que <html>, <head>, et <body>.
  • La définition des titres de section avec les balises <h1> à <h6>.
  • La création de paragraphes avec la balise <p>.
  • L'ajout d'images avec la balise <img>.
  • La création de liens hypertextes avec la balise <a>.
  • La structuration de listes avec les balises <ul>, <ol>, et <li>.
  • La création de tableaux avec les balises <table>, <tr>, et <td>.
  • L'ajout de formulaires avec la balise <form>.

Exemple

Voici un exemple simple de code HTML qui affiche un titre, un paragraphe, et une image :


<!DOCTYPE html>
<html lang="fr">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Exemple HTML</title>
</head>
<body>
    <h1>Bienvenue sur notre site web</h1>
    <p>Ceci est un paragraphe d'exemple.</p>
    <img src="exemple.jpg" alt="Image d'exemple">
</body>
</html>
