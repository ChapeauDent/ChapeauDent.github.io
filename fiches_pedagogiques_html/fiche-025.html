Auteur : Raymond Nicolas

Compétence : Utiliser les entités HTML ( , <, ©, etc.)

Date : 21/06/2025

Version : v1.0

Objectif

Comprendre l'utilisation des entités HTML pour afficher des caractères spéciaux.

Théorie

Les entités HTML sont utilisées pour afficher des caractères spéciaux qui ont une signification particulière en HTML. Elles commencent par un "&" et se terminent par un ";". Voici quelques exemples courants :

  • &nbsp; : espace insécable
  • &lt; : signe inférieur (<)
  • &gt; : signe supérieur (>)
  • &amp; : esperluette (&)
  • &quot; : guillemet double (")
  • &apos; : guillemet simple (')
  • &copy; : symbole de copyright (©)
  • &reg; : symbole de marque déposée (®)

Exemple

Voici un exemple de code HTML utilisant des entités :


<!DOCTYPE html>
<html lang="fr">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Exemple d'entités HTML</title>
</head>
<body>
    <p>Espace insécable : &nbsp;</p>
    <p>Signe inférieur : &lt;</p>
    <p>Signe supérieur : &gt;</p>
    <p>Esperluette : &amp;</p>
    <p>Guillemet double : &quot;</p>
    <p>Guillemet simple : &apos;</p>
    <p>Symbole de copyright : &copy;</p>
    <p>Symbole de marque déposée : &reg;</p>
</body>
</html>
