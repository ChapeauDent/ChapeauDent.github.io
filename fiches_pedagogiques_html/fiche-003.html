Compétence : Ajouter des attributs simples

Date : 21/06/2025

Version : v1.0

Objectif

Apprendre à ajouter des attributs simples aux éléments HTML.

Théorie

Les attributs HTML sont utilisés pour fournir des informations supplémentaires sur les éléments HTML. Ils sont toujours inclus dans la balise d'ouverture et se présentent sous la forme de paires nom-valeur. Voici quelques attributs simples couramment utilisés :

  • id : Un identifiant unique pour l'élément.
  • class : Une classe CSS pour l'élément.
  • src : L'URL de la source d'un élément (par exemple, une image).
  • alt : Un texte alternatif pour une image.
  • href : L'URL d'un lien.
  • title : Un texte d'information sur l'élément (affiché comme infobulle).

Exemple

Voici un exemple simple de code HTML utilisant des attributs :


<!DOCTYPE html>
<html lang="fr">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Exemple d'attributs HTML</title>
</head>
<body>
    <h1 id="titre-principal" class="titre">Bienvenue sur mon site web</h1>
    <p class="description">Ceci est un paragraphe avec un <a href="https://example.com" title="Lien vers example.com">lien</a>.</p>
    <img src="image.jpg" alt="Description de l'image" title="Image d'exemple">
</body>
</html>
