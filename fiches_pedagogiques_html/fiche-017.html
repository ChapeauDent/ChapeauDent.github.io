Compétence : Ajouter un titre à une page

Date : 21/06/2025

Version : v1.0

Objectif

Apprendre à ajouter un titre à une page HTML en utilisant l'élément <title>.

Théorie

L'élément <title> est utilisé pour définir le titre d'une page HTML. Ce titre est affiché dans la barre de titre du navigateur, ainsi que dans les résultats des moteurs de recherche. Il est important de choisir un titre pertinent et descriptif pour améliorer le référencement de la page.

Le titre doit être placé à l'intérieur de l'élément <head> de la page HTML. Voici un exemple de structure de document HTML avec un titre :

Exemple


<!DOCTYPE html>
<html lang="fr">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Mon Titre de Page</title>
</head>
<body>
    <h1>Bienvenue sur ma page</h1>
    <p>Ceci est un exemple de page HTML avec un titre.</p>
</body>
</html>
