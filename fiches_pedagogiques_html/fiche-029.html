Compétence : Utiliser des éléments vides (<br>, <hr>, <img>)

Date : 20/06/2025

Version : v1.0

Objectif

Comprendre l'utilisation des éléments vides en HTML.

Théorie

Les éléments vides en HTML sont des balises qui n'ont pas de contenu. Ils sont utilisés pour insérer des éléments spécifiques dans une page web, tels que des sauts de ligne, des lignes horizontales et des images. Voici quelques exemples d'éléments vides :

  • <br> : Utilisé pour insérer un saut de ligne.
  • <hr> : Utilisé pour insérer une ligne horizontale.
  • <img> : Utilisé pour insérer une image.

Exemple

Voici un exemple simple de code HTML utilisant des éléments vides :

<!DOCTYPE html>
<html lang="fr">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Exemple d'Éléments Vides</title>
</head>
<body>
    <p>Ceci est un paragraphe.<br>Ceci est une nouvelle ligne dans le même paragraphe.</p>
    <hr>
    <p>Voici une image :</p>
    <img src="exemple.jpg" alt="Exemple d'image">
</body>
</html>
