Compétence : Utiliser l’élément <form>

Date : 21/06/2025

Version : v1.0

Objectif

Comprendre l'utilisation de l'élément <form> en HTML pour créer des formulaires interactifs.

Théorie

L'élément <form> en HTML est utilisé pour créer des formulaires permettant aux utilisateurs de saisir et de soumettre des données. Les formulaires sont essentiels pour les interactions utilisateur sur les sites web, comme les inscriptions, les connexions, les recherches, etc.

Un formulaire HTML peut contenir divers éléments de saisie, tels que :

  • <input> : pour les champs de texte, les cases à cocher, les boutons radio, etc.
  • <textarea> : pour les zones de texte multilignes.
  • <select> : pour les listes déroulantes.
  • <button> : pour les boutons de soumission ou de réinitialisation.
  • <label> : pour associer des étiquettes aux éléments de formulaire.

Les attributs courants de l'élément <form> incluent :

  • action : l'URL vers laquelle les données du formulaire sont envoyées lors de la soumission.
  • method : la méthode HTTP utilisée pour envoyer les données (généralement "GET" ou "POST").
  • enctype : le type d'encodage des données lors de l'envoi (utile pour les fichiers).

Exemple

Voici un exemple simple de formulaire HTML :


<!DOCTYPE html>
<html lang="fr">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Exemple de Formulaire</title>
</head>
<body>
    <form action="/submit" method="post">
        <label for="nom">Nom :</label>
        <input type="text" id="nom" name="nom"><br><br>
        <label for="email">Email :</label>
        <input type="email" id="email" name="email"><br><br>
        <input type="submit" value="Envoyer">
    </form>
</body>
</html>
