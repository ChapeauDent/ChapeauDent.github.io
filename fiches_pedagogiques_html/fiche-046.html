Compétence : Accessibilité (a11y) : utiliser les attributs aria-*, les rôles (role="button", etc.)

Date : 20/06/2025

Version : v1.0

Objectif

Comprendre l'importance de l'accessibilité en HTML et apprendre à utiliser les attributs aria-* et les rôles pour améliorer l'accessibilité des pages web.

Théorie

L'accessibilité (a11y) est une pratique essentielle dans le développement web qui vise à rendre les sites web utilisables par le plus grand nombre de personnes possible, y compris les personnes en situation de handicap. Les attributs ARIA (Accessible Rich Internet Applications) et les rôles permettent d'améliorer l'accessibilité des éléments HTML en fournissant des informations supplémentaires aux technologies d'assistance.

Les attributs ARIA courants incluent :

  • aria-label : Fournit une étiquette accessible pour un élément.
  • aria-hidden : Indique si un élément est visible ou non pour les technologies d'assistance.
  • aria-live : Indique la priorité des mises à jour dynamiques dans une région.

Les rôles ARIA courants incluent :

  • role="button" : Indique qu'un élément agit comme un bouton.
  • role="navigation" : Indique une région de navigation.
  • role="main" : Indique la région principale du contenu.

Exemple

Voici un exemple de code HTML utilisant les attributs ARIA et les rôles :

<!DOCTYPE html>
<html lang="fr">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Exemple d'Accessibilité</title>
</head>
<body>
    <nav role="navigation" aria-label="Menu principal">
        <ul>
            <li><a href="#section1">Section 1</a></li>
            <li><a href="#section2">Section 2</a></li>
        </ul>
    </nav>
    <main role="main">
        <section id="section1" aria-labelledby="titre1">
            <h2 id="titre1">Titre de la Section 1</h2>
            <p>Contenu de la section 1.</p>
        </section>
        <section id="section2" aria-labelledby="titre2">
            <h2 id="titre2">Titre de la Section 2</h2>
            <p>Contenu de la section 2.</p>
            <button role="button" aria-pressed="false">Cliquez-moi</button>
        </section>
    </main>
</body>
</html>
