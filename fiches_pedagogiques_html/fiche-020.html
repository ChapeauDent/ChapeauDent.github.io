Compétence : Utiliser l’attribut href dans un lien

Date : 21/06/2025

Version : v1.0

Objectif

Comprendre comment utiliser l’attribut href dans un lien en HTML.

Théorie

L’attribut href (Hypertext REFerence) est utilisé pour spécifier la destination d’un lien hypertexte en HTML. Il est utilisé avec l’élément <a> (ancre) pour créer des liens vers d’autres pages web, des fichiers, des adresses email, ou des ancres internes dans la même page.

Voici quelques utilisations courantes de l’attribut href :

  • Lien vers une autre page web : <a href="https://www.example.com">Visitez Example</a>
  • Lien vers un fichier : <a href="documents/rapport.pdf">Téléchargez le rapport</a>
  • Lien vers une adresse email : <a href="mailto:contact@example.com">Envoyez un email</a>
  • Lien vers une ancre interne : <a href="#section1">Aller à la section 1</a>

Exemple

Voici un exemple de code HTML utilisant l’attribut href :


<!DOCTYPE html>
<html lang="fr">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Exemple de lien avec href</title>
</head>
<body>
    <a href="https://www.example.com">Visitez Example</a>
    <br>
    <a href="documents/rapport.pdf">Téléchargez le rapport</a>
    <br>
    <a href="mailto:contact@example.com">Envoyez un email</a>
    <br>
    <a href="#section1">Aller à la section 1</a>
    <h2 id="section1">Section 1</h2>
</body>
</html>
