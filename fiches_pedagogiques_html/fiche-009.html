Compétence : Utiliser l’élément <a>

Date : 21/06/2025

Version : v1.0

Objectif

Comprendre l'utilisation de l'élément <a> pour créer des liens hypertextes en HTML.

Théorie

L'élément <a> (ancre) est utilisé pour créer des liens hypertextes, qui permettent de naviguer d'une page à une autre ou de se déplacer à l'intérieur d'une même page. L'attribut principal de l'élément <a> est l'attribut href, qui spécifie la destination du lien.

Voici quelques utilisations courantes de l'élément <a> :

  • Créer un lien vers une autre page web : <a href="https://www.example.com">Visitez Example</a>
  • Créer un lien vers une section de la même page : <a href="#section1">Aller à la section 1</a>
  • Créer un lien pour envoyer un e-mail : <a href="mailto:example@example.com">Envoyer un e-mail</a>
  • Ouvrir un lien dans un nouvel onglet : <a href="https://www.example.com" target="_blank">Ouvrir dans un nouvel onglet</a>

Exemple

Voici un exemple simple de code HTML utilisant l'élément <a> :


<!DOCTYPE html>
<html lang="fr">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Exemple de lien</title>
</head>
<body>
    <a href="https://www.example.com">Visitez Example</a>
    <br>
    <a href="#section1">Aller à la section 1</a>
    <br>
    <a href="mailto:example@example.com">Envoyer un e-mail</a>
    <br>
    <a href="https://www.example.com" target="_blank">Ouvrir dans un nouvel onglet</a>
</body>
</html>
