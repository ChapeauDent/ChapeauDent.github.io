Compétence : Ajouter une vidéo (<video>)

Date : 20/06/2025

Version : v1.0

Objectif

Apprendre à ajouter une vidéo dans une page HTML en utilisant la balise <video>.

Théorie

La balise <video> permet d'intégrer des vidéos dans une page web. Elle offre plusieurs attributs pour contrôler la lecture, le volume, l'affichage des contrôles, etc. Voici quelques attributs courants :

  • src : spécifie l'URL de la vidéo.
  • controls : affiche les contrôles de lecture (play, pause, volume, etc.).
  • autoplay : démarre la lecture de la vidéo automatiquement.
  • loop : rejoue la vidéo en boucle.
  • muted : démarre la vidéo en mode silencieux.

Exemple

Voici un exemple simple de code HTML pour ajouter une vidéo avec des contrôles :

<!DOCTYPE html>
<html lang="fr">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Exemple Vidéo</title>
</head>
<body>
    <video src="video.mp4" controls>
        Votre navigateur ne supporte pas la balise <video>.
    </video>
</body>
</html>
