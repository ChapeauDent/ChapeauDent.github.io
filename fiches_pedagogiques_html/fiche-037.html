Compétence : Utiliser les balises <fieldset> et <legend> dans les formulaires

Date : 20/06/2025

Version : v1.0

Objectif

Comprendre l'utilisation des balises <fieldset> et <legend> dans les formulaires HTML.

Théorie

Les balises <fieldset> et <legend> sont utilisées pour regrouper des éléments de formulaire et fournir une légende descriptive. Cela permet de structurer et d'organiser les formulaires de manière sémantique et accessible.

  • <fieldset> : Utilisée pour regrouper des éléments de formulaire.
  • <legend> : Utilisée pour fournir une légende descriptive pour le groupe d'éléments de formulaire.

Exemple

<!DOCTYPE html>
<html lang="fr">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Exemple Fieldset et Legend</title>
</head>
<body>
    <form>
        <fieldset>
            <legend>Informations Personnelles</legend>
            <label for="name">Nom :</label>
            <input type="text" id="name" name="name"><br>
            <label for="email">Email :</label>
            <input type="email" id="email" name="email">
        </fieldset>
    </form>
</body>
</html>
