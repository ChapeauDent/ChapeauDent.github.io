Compétence : Créer une structure de document

Date : 21/06/2025

Version : v1.0

Objectif

Comprendre comment créer une structure de document HTML.

Théorie

La structure de base d'un document HTML comprend les éléments suivants :

  • <!DOCTYPE html> : Déclare le type de document HTML.
  • <html> : Élément racine qui contient tout le contenu HTML.
  • <head> : Contient les métadonnées du document, telles que le titre, les liens vers les feuilles de style, les scripts, etc.
  • <title> : Définit le titre du document affiché dans l'onglet du navigateur.
  • <meta> : Fournit des métadonnées supplémentaires, telles que le jeu de caractères et la description.
  • <body> : Contient le contenu visible de la page, tel que les paragraphes, les titres, les images, les liens, etc.

Exemple

Voici un exemple de structure de document HTML :


<!DOCTYPE html>
<html lang="fr">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Exemple de Structure HTML</title>
</head>
<body>
    <h1>Bienvenue sur notre site web</h1>
    <p>Ceci est un exemple de structure de document HTML.</p>
</body>
</html>
