Compétence : Utiliser un élément modifiant la mise en forme du contenu

Date : 21/06/2025

Version : v1.0

Objectif

Apprendre à utiliser les éléments HTML qui modifient la mise en forme du contenu.

Théorie

Les éléments HTML permettent de modifier la mise en forme du contenu de différentes manières. Voici quelques-uns des éléments les plus couramment utilisés :

  • <strong> : Utilisé pour rendre le texte en gras.
  • <em> : Utilisé pour rendre le texte en italique.
  • <mark> : Utilisé pour surligner le texte.
  • <small> : Utilisé pour réduire la taille du texte.
  • <del> : Utilisé pour barrer le texte.
  • <ins> : Utilisé pour souligner le texte.
  • <sub> : Utilisé pour afficher le texte en indice.
  • <sup> : Utilisé pour afficher le texte en exposant.

Exemple

Voici un exemple de code HTML utilisant ces éléments :


<p>Texte normal</p>
<p><strong>Texte en gras</strong></p>
<p><em>Texte en italique</em></p>
<p><mark>Texte surligné</mark></p>
<p><small>Texte réduit</small></p>
<p><del>Texte barré</del></p>
<p><ins>Texte souligné</ins></p>
<p><sub>Texte en indice</sub></p>
<p><sup>Texte en exposant</sup></p>
