Auteur : Raymond Nicolas

Compétence : Identifier les structures HTML (hiérarchie, indentation)

Date : 20/06/2025

Version : v1.0

Objectif

Comprendre comment identifier et structurer correctement le code HTML en utilisant la hiérarchie et l'indentation.

Théorie

La structure HTML est essentielle pour organiser le contenu d'une page web de manière logique et lisible. Une bonne hiérarchie et une indentation correcte permettent de :

  • Améliorer la lisibilité du code pour les développeurs.
  • Faciliter la maintenance et la mise à jour du code.
  • Assurer une structure sémantique correcte pour les moteurs de recherche et les technologies d'assistance.

Voici quelques éléments clés de la structure HTML :

  • Les balises HTML doivent être imbriquées correctement pour refléter la hiérarchie du contenu.
  • Utiliser des balises sémantiques appropriées (<header>, <main>, <footer>, etc.) pour décrire la structure du document.
  • Indenter le code de manière cohérente pour améliorer la lisibilité.

Exemple

Voici un exemple de structure HTML correctement hiérarchisée et indentée :


<!DOCTYPE html>
<html lang="fr">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Exemple de Structure HTML</title>
</head>
<body>
    <header>
        <h1>Titre de la Page</h1>
        <nav>
            <ul>
                <li><a href="#section1">Section 1</a></li>
                <li><a href="#section2">Section 2</a></li>
            </ul>
        </nav>
    </header>
    <main>
        <section id="section1">
            <h2>Section 1</h2>
            <p>Contenu de la section 1.</p>
        </section>
        <section id="section2">
            <h2>Section 2</h2>
            <p>Contenu de la section 2.</p>
        </section>
    </main>
    <footer>
        <p>Pied de page</p>
    </footer>
</body>
</html>
