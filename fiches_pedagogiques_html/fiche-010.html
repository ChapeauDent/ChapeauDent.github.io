Compétence : Ajouter un saut de ligne à un texte

Date : 21/06/2025

Version : v1.0

Objectif

Apprendre à ajouter un saut de ligne à un texte en HTML.

Théorie

En HTML, l'élément <br> est utilisé pour insérer un saut de ligne dans le texte. Contrairement à l'élément <p> qui crée un nouveau paragraphe, l'élément <br> permet de continuer le texte sur une nouvelle ligne sans créer de séparation de paragraphe.

Voici quelques points importants concernant l'utilisation de l'élément <br> :

  • L'élément <br> est un élément vide, ce qui signifie qu'il n'a pas de contenu et ne nécessite pas de balise de fermeture.
  • Il est souvent utilisé dans les poèmes, les adresses et les autres textes où les sauts de ligne sont significatifs.
  • Il peut être utilisé plusieurs fois de suite pour créer des espaces supplémentaires entre les lignes.

Exemple

Voici un exemple simple de code HTML utilisant l'élément <br> :


<!DOCTYPE html>
<html lang="fr">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Exemple de saut de ligne</title>
</head>
<body>
    <p>Ceci est une ligne de texte.<br>Ceci est une nouvelle ligne de texte.</p>
</body>
</html>
