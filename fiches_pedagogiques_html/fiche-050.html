Compétence : Ajouter des attributs complexes (data-*)

Date : 20/06/2025

Version : v1.0

Objectif

Comprendre l'utilisation des attributs complexes data-* en HTML.

Théorie

Les attributs data-* permettent de stocker des données personnalisées dans les éléments HTML. Ils sont utilisés pour ajouter des informations supplémentaires qui peuvent être utilisées par JavaScript pour manipuler le DOM ou pour des fonctionnalités spécifiques.

Les attributs data-* suivent la syntaxe data-nom, où nom est le nom de l'attribut. Par exemple, data-id, data-role, etc.

Voici quelques utilisations courantes des attributs data-* :

  • Stocker des identifiants uniques pour les éléments.
  • Ajouter des rôles ou des types spécifiques aux éléments.
  • Stocker des informations de configuration ou des paramètres.
  • Ajouter des données contextuelles pour les scripts JavaScript.

Exemple

Voici un exemple de code HTML utilisant les attributs data-* :

<!DOCTYPE html>
<html lang="fr">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Exemple data-* en HTML</title>
</head>
<body>
    <div data-id="123" data-role="admin">
        <p>Utilisateur : Admin</p>
    </div>
    <script>
        const div = document.querySelector('div');
        console.log(div.dataset.id); // Affiche "123"
        console.log(div.dataset.role); // Affiche "admin"
    </script>
</body>
</html>
