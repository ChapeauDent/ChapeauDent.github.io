Auteur : Raymond Nicolas

Compétence : Ajouter un titre à une image (alt, title)

Date : 21/06/2025

Version : v1.0

Objectif

Apprendre à ajouter un titre à une image en utilisant les attributs alt et title en HTML.

Théorie

Les attributs alt et title sont utilisés pour fournir des informations supplémentaires sur une image en HTML :

  • alt (texte alternatif) : utilisé pour décrire le contenu de l'image. Ce texte est affiché si l'image ne peut pas être chargée et est également utilisé par les lecteurs d'écran pour les utilisateurs malvoyants.
  • title (titre) : utilisé pour fournir des informations supplémentaires sur l'image. Ce texte apparaît généralement sous forme d'infobulle lorsque l'utilisateur survole l'image avec la souris.

Exemple

Voici un exemple de code HTML qui montre comment ajouter les attributs alt et title à une image :


<!DOCTYPE html>
<html lang="fr">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Exemple d'image avec attributs alt et title</title>
</head>
<body>
    <img src="chemin/vers/image.jpg" alt="Description de l'image" title="Titre de l'image">
</body>
</html>
