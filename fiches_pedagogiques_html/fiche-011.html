Compétence : Créer une structure de page généraliste

Date : 21/06/2025

Version : v1.0

Objectif

Apprendre à créer une structure de page HTML généraliste.

Théorie

Une structure de page HTML généraliste comprend les éléments de base nécessaires pour créer une page web fonctionnelle et bien organisée. Voici les principaux éléments :

  • <!DOCTYPE html> : Déclare le type de document et la version HTML utilisée.
  • <html> : Élément racine qui contient tout le contenu de la page.
  • <head> : Contient les métadonnées, le titre de la page, les liens vers les fichiers CSS et les scripts JavaScript.
  • <body> : Contient le contenu visible de la page, comme les titres, les paragraphes, les images, les liens, etc.
  • <header> : Section d'en-tête de la page, généralement utilisée pour le titre et la navigation.
  • <nav> : Section de navigation, contenant des liens vers d'autres pages ou sections de la même page.
  • <main> : Section principale du contenu de la page.
  • <footer> : Section de pied de page, contenant des informations de copyright, des liens vers les politiques de confidentialité, etc.

Exemple

Voici un exemple de structure de page HTML généraliste :


<!DOCTYPE html>
<html lang="fr">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Page Généraliste</title>
    <link rel="stylesheet" href="style.css">
</head>
<body>
    <header>
        <h1>Bienvenue sur ma page</h1>
        <nav>
            <ul>
                <li><a href="#section1">Section 1</a></li>
                <li><a href="#section2">Section 2</a></li>
                <li><a href="#section3">Section 3</a></li>
            </ul>
        </nav>
    </header>
    <main>
        <section id="section1">
            <h2>Section 1</h2>
            <p>Contenu de la section 1.</p>
        </section>
        <section id="section2">
            <h2>Section 2</h2>
            <p>Contenu de la section 2.</p>
        </section>
        <section id="section3">
            <h2>Section 3</h2>
            <p>Contenu de la section 3.</p>
        </section>
    </main>
    <footer>
        <p>© 2025 Raymond Nicolas. Tous droits réservés.</p>
    </footer>
</body>
</html>
