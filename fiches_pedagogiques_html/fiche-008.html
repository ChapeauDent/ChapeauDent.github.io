Compétence : Ajouter une image à une page

Date : 21/06/2025

Version : v1.0

Objectif

Apprendre à ajouter et configurer des images dans une page HTML.

Théorie

En HTML, l'ajout d'images à une page web se fait à l'aide de la balise <img>. Cette balise est un élément vide, ce qui signifie qu'elle n'a pas de contenu de fermeture. Elle utilise plusieurs attributs pour définir l'image à afficher et ses propriétés :

  • src : L'URL de l'image à afficher. Cet attribut est obligatoire.
  • alt : Le texte alternatif à afficher si l'image ne peut pas être chargée. Cet attribut est fortement recommandé pour l'accessibilité.
  • width et height : Les dimensions de l'image. Ces attributs sont optionnels mais peuvent être utilisés pour définir la taille de l'image.
  • title : Le texte qui s'affiche lorsque l'utilisateur survole l'image avec la souris. Cet attribut est optionnel.

Voici un exemple de balise <img> :

<img src="chemin/vers/image.jpg" alt="Description de l'image" width="500" height="300" title="Titre de l'image">

Exemple

Voici un exemple complet de code HTML qui affiche une image :


<!DOCTYPE html>
<html lang="fr">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Exemple d'ajout d'image</title>
</head>
<body>
    <h1>Exemple d'ajout d'image</h1>
    <img src="chemin/vers/image.jpg" alt="Description de l'image" width="500" height="300" title="Titre de l'image">
</body>
</html>
