Compétence : Utiliser des attributs booléens (checked, disabled, required)

Date : 20/06/2025

Version : v1.0

Objectif

Comprendre l'utilisation des attributs booléens en HTML tels que checked, disabled, et required.

Théorie

Les attributs booléens en HTML sont des attributs qui n'ont pas de valeur. Leur présence indique que l'élément doit avoir un comportement particulier. Voici quelques exemples :

  • checked : Indique qu'une case à cocher ou un bouton radio doit être sélectionné par défaut.
  • disabled : Indique qu'un élément de formulaire doit être désactivé.
  • required : Indique qu'un champ de formulaire doit être rempli avant de pouvoir soumettre le formulaire.

Exemple

Voici un exemple de code HTML utilisant les attributs booléens :

<!DOCTYPE html>
<html lang="fr">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Exemple d'Attributs Booléens</title>
</head>
<body>
    <form>
        <label>
            <input type="checkbox" checked> Case à cocher sélectionnée par défaut
        </label>
        <br>
        <label>
            <input type="text" disabled> Champ de texte désactivé
        </label>
        <br>
        <label>
            <input type="text" required> Champ de texte requis
        </label>
    </form>
</body>
</html>
