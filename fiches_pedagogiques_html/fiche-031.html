Compétence : Identifier les niveaux de titres et leur usage sémantique

Date : 20/06/2025

Version : v1.0

Objectif

Comprendre les niveaux de titres en HTML et leur usage sémantique.

Théorie

En HTML, les titres sont définis par les balises <h1> à <h6>. Chaque niveau de titre a une importance sémantique différente :

  • <h1> : Titre principal de la page, généralement utilisé une seule fois.
  • <h2> : Sous-titre de premier niveau, utilisé pour les sections principales.
  • <h3> : Sous-titre de deuxième niveau, utilisé pour les sous-sections.
  • <h4> : Sous-titre de troisième niveau.
  • <h5> : Sous-titre de quatrième niveau.
  • <h6> : Sous-titre de cinquième niveau, rarement utilisé.

Les titres permettent de structurer le contenu de manière hiérarchique et d'améliorer l'accessibilité et le référencement des pages web.

Exemple


<!DOCTYPE html>
<html lang="fr">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Exemple de Titres HTML</title>
</head>
<body>
    <h1>Titre Principal</h1>
    <h2>Section Principale</h2>
    <h3>Sous-section</h3>
    <h4>Sous-sous-section</h4>
    <h5>Sous-sous-sous-section</h5>
    <h6>Sous-sous-sous-sous-section</h6>
</body>
</html>
