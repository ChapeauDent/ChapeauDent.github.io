Compétence : Créer des liens vers des ancres internes

Date : 20/06/2025

Version : v1.0

Objectif

Comprendre comment créer des liens vers des ancres internes dans une page HTML.

Théorie

Les ancres internes permettent de créer des liens qui mènent à une section spécifique de la même page. Cela est particulièrement utile pour les longues pages avec beaucoup de contenu. Pour créer une ancre interne, vous devez :

  • Définir une cible d'ancre en utilisant l'attribut id sur un élément HTML.
  • Créer un lien qui pointe vers cette cible en utilisant l'attribut href avec le symbole # suivi de l'identifiant de la cible.

Exemple

Voici un exemple de code HTML qui montre comment créer des liens vers des ancres internes :

<!DOCTYPE html>
<html lang="fr">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Exemple d'ancres internes</title>
</head>
<body>
    <h1 id="section1">Section 1</h1>
    <p>Ceci est la section 1.</p>
    <a href="#section2">Aller à la section 2</a>

    <h1 id="section2">Section 2</h1>
    <p>Ceci est la section 2.</p>
    <a href="#section1">Retour à la section 1</a>
</body>
</html>
