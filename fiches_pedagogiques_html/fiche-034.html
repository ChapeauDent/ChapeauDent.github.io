Compétence : Ajouter des intitulés aux champs de formulaire

Date : 20/06/2025

Version : v1.0

Objectif

Comprendre comment ajouter des intitulés aux champs de formulaire en HTML avec la balise <label>.

Théorie

La balise <label> en HTML est utilisée pour associer un intitulé à un champ de formulaire. Cela permet de rendre les formulaires plus accessibles et plus faciles à utiliser. En associant un <label> à un champ de formulaire, les utilisateurs peuvent cliquer sur l'intitulé pour activer le champ correspondant.

Pour associer un <label> à un champ de formulaire, il suffit de définir l'attribut for du <label> avec l'identifiant (id) du champ de formulaire correspondant.

Exemple

<!DOCTYPE html>
<html lang="fr">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Exemple de Formulaire</title>
</head>
<body>
    <form>
        <label for="nom">Nom :</label>
        <input type="text" id="nom" name="nom">
        <br>
        <label for="email">Email :</label>
        <input type="email" id="email" name="email">
    </form>
</body>
</html>
