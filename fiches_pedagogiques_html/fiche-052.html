Compétence : Ajouter une image vectorielle (<svg>)

Date : 20/06/2025

Version : v1.0

Objectif

Comprendre comment ajouter une image vectorielle en utilisant la balise <svg> en HTML.

Théorie

La balise <svg> permet d'intégrer des images vectorielles dans une page HTML. Les images vectorielles sont des graphiques basés sur des formes géométriques (lignes, cercles, rectangles, etc.) et peuvent être redimensionnées sans perte de qualité. Voici quelques avantages de l'utilisation des images vectorielles :

  • Redimensionnement sans perte de qualité.
  • Possibilité de manipuler les éléments graphiques avec CSS et JavaScript.
  • Meilleure performance pour les graphiques simples.

Exemple

Voici un exemple simple de code SVG qui dessine un cercle rouge :

<svg width="100" height="100" xmlns="http://www.w3.org/2000/svg">
    <circle cx="50" cy="50" r="40" stroke="black" stroke-width="3" fill="red" />
</svg>
