Compétence : Afficher une liste ordonnée (<ol>)

Date : 20/06/2025

Version : v1.0

Objectif

Comprendre comment afficher une liste ordonnée en HTML en utilisant la balise <ol>.

Théorie

La balise <ol> en HTML est utilisée pour créer des listes ordonnées. Chaque élément de la liste est représenté par la balise <li>. Les listes ordonnées sont généralement utilisées lorsque l'ordre des éléments est important, comme dans les instructions ou les étapes d'un processus.

Voici un exemple de liste ordonnée :

  • Étape 1 : Préparer les ingrédients.
  • Étape 2 : Mélanger les ingrédients.
  • Étape 3 : Cuire au four.
  • Étape 4 : Servir.

Exemple

Voici un exemple de code HTML pour créer une liste ordonnée :

<!DOCTYPE html>
<html lang="fr">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Exemple de Liste Ordonnée</title>
</head>
<body>
    <ol>
        <li>Étape 1 : Préparer les ingrédients.</li>
        <li>Étape 2 : Mélanger les ingrédients.</li>
        <li>Étape 3 : Cuire au four.</li>
        <li>Étape 4 : Servir.</li>
    </ol>
</body>
</html>
