Compétence : Ajouter un titre de section à une page

Date : 21/06/2025

Version : v1.0

Objectif

Apprendre à ajouter un titre de section à une page HTML en utilisant les balises appropriées.

Théorie

Les titres de section en HTML sont définis à l'aide des balises de titre, allant de <h1> à <h6>. Ces balises permettent de structurer le contenu de la page en différentes sections, chaque titre ayant un niveau d'importance décroissant :

  • <h1> : Titre de niveau 1, le plus important
  • <h2> : Titre de niveau 2
  • <h3> : Titre de niveau 3
  • <h4> : Titre de niveau 4
  • <h5> : Titre de niveau 5
  • <h6> : Titre de niveau 6, le moins important

Il est important de structurer correctement les titres pour améliorer la lisibilité et l'accessibilité de la page web.

Exemple

Voici un exemple de code HTML avec différents niveaux de titres :


<!DOCTYPE html>
<html lang="fr">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Exemple de Titres</title>
</head>
<body>
    <h1>Titre de niveau 1</h1>
    <h2>Titre de niveau 2</h2>
    <h3>Titre de niveau 3</h3>
    <h4>Titre de niveau 4</h4>
    <h5>Titre de niveau 5</h5>
    <h6>Titre de niveau 6</h6>
</body>
</html>
