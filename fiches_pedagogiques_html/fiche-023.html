Compétence : Ajouter un formulaire avec des champs à une page

Date : 21/06/2025

Version : v1.0

Objectif

Apprendre à ajouter un formulaire avec des champs à une page HTML.

Théorie

Les formulaires HTML permettent de collecter des données auprès des utilisateurs. Ils sont utilisés pour diverses interactions, telles que les inscriptions, les connexions, les recherches, etc. Un formulaire HTML est défini à l'aide de l'élément <form>, et les champs de formulaire sont créés à l'aide de divers éléments comme <input>, <textarea>, <select>, etc.

Les principaux éléments de formulaire incluent :

  • <input> : Utilisé pour créer des champs de saisie de texte, des cases à cocher, des boutons radio, des boutons de soumission, etc.
  • <textarea> : Utilisé pour créer des zones de texte multilignes.
  • <select> : Utilisé pour créer des listes déroulantes.
  • <label> : Utilisé pour associer des étiquettes aux champs de formulaire.

Exemple

Voici un exemple de code HTML pour ajouter un formulaire avec des champs à une page :


<!DOCTYPE html>
<html lang="fr">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Exemple de Formulaire</title>
</head>
<body>
    <form action="/submit" method="post">
        <label for="nom">Nom :</label>
        <input type="text" id="nom" name="nom"><br><br>
        <label for="email">Email :</label>
        <input type="email" id="email" name="email"><br><br>
        <label for="message">Message :</label><br>
        <textarea id="message" name="message" rows="4" cols="50"></textarea><br><br>
        <input type="submit" value="Envoyer">
    </form>
</body>
</html>
