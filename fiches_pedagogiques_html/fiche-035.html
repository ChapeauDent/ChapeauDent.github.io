Compétence : Structurer une page avec des balises sémantiques HTML5

Date : 20/06/2025

Version : v1.0

Objectif

Comprendre comment structurer une page web en utilisant les balises sémantiques HTML5.

Théorie

Les balises sémantiques HTML5 permettent de donner du sens au contenu d'une page web. Elles aident les moteurs de recherche et les technologies d'assistance à comprendre la structure et le contenu de la page. Voici quelques-unes des balises sémantiques les plus courantes :

  • <header> : représente l'en-tête d'une section ou d'une page.
  • <nav> : représente une section de navigation.
  • <main> : représente le contenu principal du document.
  • <article> : représente un contenu autonome qui pourrait être distribué indépendamment.
  • <section> : représente une section thématique du contenu.
  • <aside> : représente du contenu lié mais indépendant du contenu principal.
  • <footer> : représente le pied de page d'une section ou d'une page.

Exemple

Voici un exemple de structure de page utilisant les balises sémantiques HTML5 :

<!DOCTYPE html>
<html lang="fr">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Exemple de Structure Sémantique</title>
</head>
<body>
    <header>
        <h1>Titre de la Page</h1>
    </header>
    <nav>
        <ul>
            <li><a href="#section1">Section 1</a></li>
            <li><a href="#section2">Section 2</a></li>
        </ul>
    </nav>
    <main>
        <article>
            <section id="section1">
                <h2>Section 1</h2>
                <p>Contenu de la section 1.</p>
            </section>
            <section id="section2">
                <h2>Section 2</h2>
                <p>Contenu de la section 2.</p>
            </section>
        </article>
        <aside>
            <p>Contenu additionnel ou publicité.</p>
        </aside>
    </main>
    <footer>
        <p>Pied de page de la page.</p>
    </footer>
</body>
</html>
