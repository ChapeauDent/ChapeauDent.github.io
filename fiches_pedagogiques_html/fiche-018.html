Compétence : Ajouter un paragraphe (<p>)

Date : 21/06/2025

Version : v1.0

Objectif

Comprendre comment ajouter et utiliser des paragraphes en HTML.

Théorie

Le paragraphe en HTML est représenté par l'élément <p>. Il est utilisé pour définir un bloc de texte. Les paragraphes sont des éléments de bloc, ce qui signifie qu'ils occupent toute la largeur disponible et commencent sur une nouvelle ligne.

Voici quelques points importants concernant l'utilisation des paragraphes :

  • Les paragraphes sont utilisés pour structurer le texte en sections distinctes.
  • Ils peuvent contenir du texte, des liens, des images, et d'autres éléments en ligne.
  • Les paragraphes peuvent être stylisés avec CSS pour modifier leur apparence.

Exemple

Voici un exemple simple de code HTML utilisant des paragraphes :


<!DOCTYPE html>
<html lang="fr">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Exemple de Paragraphe</title>
</head>
<body>
    <p>Ceci est un paragraphe de texte.</p>
    <p>Les paragraphes sont utilisés pour structurer le contenu textuel.</p>
</body>
</html>
