Objectif

Comprendre l'importance et l'utilisation de la déclaration de type de document (DOCTYPE) en HTML.

Théorie

La déclaration <!DOCTYPE html> est utilisée pour définir la version du HTML utilisée dans le document. Elle doit être placée au tout début du document HTML, avant la balise <html>. Cette déclaration permet aux navigateurs web de rendre correctement la page en mode standard, évitant ainsi le mode de compatibilité qui pourrait entraîner des comportements imprévisibles.

En HTML5, la déclaration DOCTYPE est simplifiée et se présente sous la forme suivante :

<!DOCTYPE html>

Cette déclaration est suffisante pour indiquer que le document utilise la norme HTML5.

Exemple

Voici un exemple de document HTML5 complet avec la déclaration DOCTYPE :

<!DOCTYPE html>
<html lang="fr">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Exemple de DOCTYPE</title>
</head>
<body>
    <h1>Bienvenue sur notre site web</h1>
    <p>Ceci est un exemple de document HTML5 avec la déclaration DOCTYPE.</p>
</body>
</html>
