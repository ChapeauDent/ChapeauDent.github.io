Compétence : Choisir l’élément pertinent selon le contenu

Date : 20/06/2025

Version : v1.0

Objectif

Comprendre comment choisir l’élément HTML pertinent selon le contenu.

Théorie

En HTML, il est important de choisir les balises sémantiques appropriées pour structurer le contenu de manière logique et significative. Voici quelques balises couramment utilisées :

  • <article> : Utilisé pour représenter un contenu autonome qui pourrait être distribué indépendamment, comme un article de blog.
  • <section> : Utilisé pour regrouper des contenus thématiquement liés, comme une section dans un chapitre.
  • <aside> : Utilisé pour représenter du contenu en marge, comme des notes ou des publicités.
  • <nav> : Utilisé pour représenter une section de navigation.
  • <header> : Utilisé pour représenter l’en-tête d’une section ou d’un article.
  • <footer> : Utilisé pour représenter le pied de page d’une section ou d’un article.

Exemple


<!DOCTYPE html>
<html lang="fr">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Exemple de balises sémantiques</title>
</head>
<body>
    <header>
        <h1>Bienvenue sur mon site</h1>
        <nav>
            <ul>
                <li><a href="#section1">Section 1</a></li>
                <li><a href="#section2">Section 2</a></li>
            </ul>
        </nav>
    </header>
    <main>
        <article>
            <section id="section1">
                <h2>Section 1</h2>
                <p>Contenu de la section 1.</p>
            </section>
            <section id="section2">
                <h2>Section 2</h2>
                <p>Contenu de la section 2.</p>
            </section>
        </article>
        <aside>
            <p>Contenu en marge.</p>
        </aside>
    </main>
    <footer>
        <p>Pied de page</p>
    </footer>
</body>
</html>
