Compétence : Créer un tableau simple

Date : 21/06/2025

Version : v1.0

Objectif

Apprendre à créer un tableau simple en HTML.

Théorie

Les tableaux en HTML permettent de structurer des données sous forme de grille. Ils sont utiles pour afficher des informations tabulaires de manière organisée. Un tableau HTML est défini par l'élément <table>, et il est composé de lignes (<tr>) et de cellules (<td> pour les données et <th> pour les en-têtes).

Voici les éléments de base d'un tableau HTML :

  • <table> : Définit le tableau.
  • <tr> : Définit une ligne dans le tableau.
  • <td> : Définit une cellule de données.
  • <th> : Définit une cellule d'en-tête.

Exemple

Voici un exemple de code HTML pour créer un tableau simple :


<!DOCTYPE html>
<html lang="fr">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Exemple de Tableau HTML</title>
</head>
<body>
    <table border="1">
        <tr>
            <th>Nom</th>
            <th>Âge</th>
            <th>Ville</th>
        </tr>
        <tr>
            <td>Alice</td>
            <td>30</td>
            <td>Paris</td>
        </tr>
        <tr>
            <td>Bob</td>
            <td>25</td>
            <td>Lyon</td>
        </tr>
    </table>
</body>
</html>
