Compétence : Regrouper les cellules d’un tableau (<thead>, <tbody>, <tfoot>)

Date : 20/06/2025

Version : v1.0

Objectif

Comprendre comment regrouper les cellules d’un tableau HTML en utilisant les balises <thead>, <tbody> et <tfoot>.

Théorie

Les balises <thead>, <tbody> et <tfoot> sont utilisées pour regrouper les cellules d’un tableau HTML en différentes sections :

  • <thead> : Définit l’en-tête du tableau.
  • <tbody> : Définit le corps principal du tableau.
  • <tfoot> : Définit le pied de tableau.

Ces balises permettent de structurer le tableau de manière sémantique et d’améliorer l’accessibilité et la lisibilité du code HTML.

Exemple

<table>
    <thead>
        <tr>
            <th>Nom</th>
            <th>Âge</th>
            <th>Ville</th>
        </tr>
    </thead>
    <tbody>
        <tr>
            <td>Alice</td>
            <td>30</td>
            <td>Paris</td>
        </tr>
        <tr>
            <td>Bob</td>
            <td>25</td>
            <td>Lyon</td>
        </tr>
    </tbody>
    <tfoot>
        <tr>
            <td colspan="3">Fin du tableau</td>
        </tr>
    </tfoot>
</table>
