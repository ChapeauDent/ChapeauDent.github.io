Compétence : Intégrer HTML avec JavaScript pour manipuler le DOM

Date : 20/06/2025

Version : v1.0

Objectif

Comprendre comment intégrer HTML avec JavaScript pour manipuler le DOM.

Théorie

Le Document Object Model (DOM) est une interface de programmation pour les documents HTML et XML. Il représente la structure de la page sous forme d'un arbre, où chaque nœud est un objet représentant une partie du document.

JavaScript peut être utilisé pour manipuler le DOM afin de modifier dynamiquement le contenu, la structure et le style des pages web. Voici quelques opérations courantes :

  • Accéder aux éléments du DOM en utilisant des méthodes telles que getElementById, getElementsByClassName, et querySelector.
  • Modifier le contenu des éléments en utilisant la propriété innerHTML ou textContent.
  • Changer les attributs des éléments en utilisant la méthode setAttribute.
  • Ajouter ou supprimer des éléments en utilisant les méthodes appendChild et removeChild.
  • Appliquer des styles en modifiant la propriété style des éléments.

Exemple

Voici un exemple simple de code JavaScript qui change le texte d'un paragraphe lorsque l'utilisateur clique sur un bouton :


<!DOCTYPE html>
<html lang="fr">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Exemple DOM</title>
    <script>
        function changeText() {
            document.getElementById("myParagraph").textContent = "Le texte a été changé!";
        }
    </script>
</head>
<body>
    <p id="myParagraph">Ceci est un paragraphe.</p>
    <button onclick="changeText()">Cliquez-moi</button>
</body>
</html>
