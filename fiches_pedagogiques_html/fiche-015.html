Compétence : Ajouter des attributs simples aux éléments HTML

Date : 21/06/2025

Version : v1.0

Objectif

Apprendre à ajouter des attributs simples aux éléments HTML pour enrichir le contenu et améliorer l'accessibilité.

Théorie

Les attributs HTML sont utilisés pour fournir des informations supplémentaires sur les éléments HTML. Ils sont ajoutés directement dans la balise d'ouverture de l'élément et sont constitués d'un nom et d'une valeur. Voici quelques attributs courants :

  • id : Un identifiant unique pour l'élément.
  • class : Une classe qui peut être utilisée pour appliquer des styles CSS.
  • src : L'URL de la source d'une image, vidéo ou autre média.
  • alt : Un texte alternatif pour une image.
  • href : L'URL vers laquelle un lien pointe.
  • title : Un texte qui apparaît comme une info-bulle lorsque l'utilisateur survole l'élément.

Les attributs sont ajoutés dans la balise d'ouverture de l'élément, par exemple :

<img src="image.jpg" alt="Description de l'image">

Exemple

Voici un exemple de code HTML utilisant différents attributs :


<!DOCTYPE html>
<html lang="fr">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Exemple d'attributs HTML</title>
</head>
<body>
    <h1 id="titre-principal" class="titre">Bienvenue sur notre site</h1>
    <p title="Paragraphe d'introduction">Ceci est un paragraphe avec un attribut title.</p>
    <a href="https://www.example.com" class="lien">Visitez notre site</a>
    <img src="image.jpg" alt="Description de l'image" class="image">
</body>
</html>
