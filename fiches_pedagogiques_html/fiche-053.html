Compétence : Utiliser les balises <template> et <slot> (préparation à l’usage de Web Components)

Date : 20/06/2025

Version : v1.0

Objectif

Comprendre l'utilisation des balises <template> et <slot> en HTML dans le contexte de la préparation à l’usage des Web Components.

Théorie

Les balises <template> et <slot> sont des éléments clés pour la création de Web Components en HTML. Elles permettent de définir des structures HTML réutilisables et de les insérer dynamiquement dans le DOM.

La balise <template> est utilisée pour déclarer des fragments de code HTML qui ne sont pas rendus immédiatement. Ces fragments peuvent être clonés et insérés dans le DOM à la demande.

La balise <slot> est utilisée pour définir des emplacements où le contenu peut être inséré dans un Web Component. Elle permet de créer des composants réutilisables avec des zones de contenu personnalisables.

Exemple

Voici un exemple simple de code HTML utilisant les balises <template> et <slot> :

<!DOCTYPE html>
<html lang="fr">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Exemple Web Component</title>
</head>
<body>

<template id="my-template">
    <div>
        <h2>Titre du composant</h2>
        <slot name="content">Contenu par défaut</slot>
    </div>
</template>

<script>
    class MyComponent extends HTMLElement {
        constructor() {
            super();
            const template = document.getElementById('my-template').content.cloneNode(true);
            this.attachShadow({ mode: 'open' }).appendChild(template);
        }
    }

    customElements.define('my-component', MyComponent);
</script>

<my-component>
    <div slot="content">Contenu personnalisé</div>
</my-component>

</body>
</html>
