Compétence : Créer un élément conteneur en ligne

Date : 21/06/2025

Version : v1.0

Objectif

Comprendre comment créer un élément conteneur en ligne en HTML.

Théorie

En HTML, un élément conteneur en ligne est utilisé pour regrouper des éléments sans interrompre le flux du texte. Contrairement aux éléments de bloc, les éléments en ligne ne commencent pas sur une nouvelle ligne et ne prennent pas toute la largeur disponible.

Le principal élément conteneur en ligne est l'élément . Il est souvent utilisé pour appliquer des styles ou des scripts à une partie spécifique du texte sans affecter le reste du contenu.

Voici quelques caractéristiques des éléments en ligne :

  • Ils ne commencent pas sur une nouvelle ligne.
  • Ils ne prennent que la largeur nécessaire pour leur contenu.
  • Ils peuvent contenir des éléments en ligne et du texte.
  • Ils peuvent être stylisés avec CSS pour changer leur apparence.

Exemple

Voici un exemple simple de code HTML utilisant l'élément :


<!DOCTYPE html>
<html lang="fr">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Exemple de conteneur en ligne</title>
    <style>
        .highlight {
            background-color: yellow;
            font-weight: bold;
        }
    </style>
</head>
<body>
    <p>Ce texte contient un <span class="highlight">élément en ligne</span> qui est stylisé.</p>
</body>
</html>
