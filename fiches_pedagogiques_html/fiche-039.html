Compétence : Différencier les éléments de bloc et en ligne

Date : 20/06/2025

Version : v1.0

Objectif

Comprendre la différence entre les éléments de bloc et les éléments en ligne en HTML.

Théorie

En HTML, les éléments peuvent être classés en deux catégories principales : les éléments de bloc et les éléments en ligne.

Éléments de bloc : Les éléments de bloc occupent toute la largeur disponible et commencent toujours sur une nouvelle ligne. Ils sont utilisés pour structurer le contenu de la page. Exemples d'éléments de bloc :

  • <div>
  • <p>
  • <h1> à <h6>
  • <ul> et <ol>
  • <table>

Éléments en ligne : Les éléments en ligne n'occupent que l'espace nécessaire à leur contenu et ne commencent pas sur une nouvelle ligne. Ils sont utilisés pour le contenu à l'intérieur des éléments de bloc. Exemples d'éléments en ligne :

  • <span>
  • <a>
  • <img>
  • <strong> et <em>
  • <input>

Exemple

Voici un exemple de code HTML utilisant des éléments de bloc et en ligne :


<!DOCTYPE html>
<html lang="fr">
<head>
    <meta charset="UTF-8">
    <title>Exemple HTML</title>
</head>
<body>
    <div>
        <h1>Titre de la page</h1>
        <p>Ceci est un paragraphe avec un <span>élément en ligne</span>.</p>
        <a href="#">Lien</a>
        <img src="image.jpg" alt="Image">
    </div>
</body>
</html>
