Compétence : Soumettre un formulaire

Date : 21/06/2025

Version : v1.0

Objectif

Comprendre comment soumettre un formulaire en HTML.

Théorie

Soumettre un formulaire en HTML implique l'utilisation de l'élément <form> avec un bouton de soumission. Le formulaire peut contenir divers champs de saisie tels que <input>, <textarea>, et <select>. Lors de la soumission, les données du formulaire sont envoyées à une URL spécifiée dans l'attribut action du formulaire.

Les principales étapes pour soumettre un formulaire sont :

  • Créer un élément <form> avec l'attribut action spécifiant l'URL de destination.
  • Ajouter des champs de saisie à l'intérieur du formulaire.
  • Ajouter un bouton de soumission avec l'attribut type="submit".

Exemple

Voici un exemple simple de formulaire HTML :


<!DOCTYPE html>
<html lang="fr">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Exemple de Formulaire</title>
</head>
<body>
    <form action="https://example.com/submit" method="post">
        <label for="nom">Nom :</label>
        <input type="text" id="nom" name="nom"><br>
        <label for="email">Email :</label>
        <input type="email" id="email" name="email"><br>
        <button type="submit">Soumettre</button>
    </form>
</body>
</html>
