Compétence : Écrire un commentaire

Date : 21/06/2025

Version : v1.0

Objectif

Apprendre à écrire des commentaires dans le code HTML pour améliorer la lisibilité et la maintenance.

Théorie

Les commentaires en HTML sont des notes ou des annotations ajoutées dans le code source pour expliquer ou décrire des parties spécifiques du code. Ils sont ignorés par les navigateurs et ne sont pas affichés sur la page web. Les commentaires sont utiles pour :

  • Expliquer le but ou la fonction d'un élément ou d'une section de code.
  • Fournir des informations supplémentaires aux développeurs qui travaillent sur le même projet.
  • Rendre le code plus lisible et plus facile à comprendre.
  • Temporarily disable parts of the code during debugging or testing.

Pour écrire un commentaire en HTML, utilisez la syntaxe suivante :

<!-- Ceci est un commentaire -->

Exemple

Voici un exemple de code HTML avec des commentaires :


<!DOCTYPE html>
<html lang="fr">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Exemple de Commentaires</title>
</head>
<body>
    <!-- Ceci est un commentaire dans le code HTML -->
    <h1>Bienvenue sur ma page web</h1>
    <p>Ceci est un paragraphe.</p>
    <!-- Fin du contenu principal -->
</body>
</html>
