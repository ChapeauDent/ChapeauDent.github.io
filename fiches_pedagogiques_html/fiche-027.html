Compétence : Utiliser les balises sémantiques de base (<main>, <header>, <footer>, <nav>)

Date : 20/06/2025

Version : v1.0

Objectif

Comprendre l'utilisation des balises sémantiques de base (<main>, <header>, <footer>, <nav>) en HTML.

Théorie

Les balises sémantiques de base en HTML permettent de structurer le contenu de manière logique et significative. Elles aident les moteurs de recherche et les technologies d'assistance à comprendre la structure de la page.

  • <header> : Représente l'en-tête de la page ou d'une section.
  • <main> : Représente le contenu principal de la page.
  • <footer> : Représente le pied de page.
  • <nav> : Représente une section de navigation.

Exemple

<!DOCTYPE html>
<html lang="fr">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Exemple Balises Sémantiques</title>
</head>
<body>
    <header>
        <h1>Bienvenue sur mon site</h1>
    </header>
    <nav>
        <ul>
            <li><a href="#section1">Section 1</a></li>
            <li><a href="#section2">Section 2</a></li>
        </ul>
    </nav>
    <main>
        <section id="section1">
            <h2>Section 1</h2>
            <p>Contenu de la section 1.</p>
        </section>
        <section id="section2">
            <h2>Section 2</h2>
            <p>Contenu de la section 2.</p>
        </section>
    </main>
    <footer>
        <p>Pied de page</p>
    </footer>
</body>
</html>
