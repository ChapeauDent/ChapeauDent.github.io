Bienvenue dans le parcours d'apprentissage de Git au sein de Code Alchimie. Nous vous proposons trois fiches détaillées qui vont vous guider pas à pas pour maîtriser les outils de gestion de versions les plus populaires. Ces cours sont conçus pour vous fournir les connaissances nécessaires pour gérer vos projets efficacement, collaborer avec d'autres développeurs et pousser vos projets sur GitHub.

Voici un aperçu des trois fiches de cours :

1. Configuration de Git

Cette fiche de cours vous aide à configurer Git sur votre machine locale. La configuration est une étape indispensable pour identifier chaque auteur des modifications sur un projet. Vous apprendrez à :

  • Configurer votre nom d'utilisateur et votre e-mail pour Git.
  • Vérifier votre configuration.
  • Comprendre l'importance de l'identification des contributeurs.

Accéder à la fiche de cours : Configuration de Git

2. Gestion des Versions avec Git

Cette fiche est une introduction aux concepts de gestion de versions avec Git. Elle explique comment suivre l'évolution de vos projets et revenir à des versions antérieures en cas de besoin. Parmi les sujets abordés :

  • Comment initialiser un nouveau repository avec git init.
  • Ajouter des fichiers avec git add.
  • Créer des commits pour enregistrer les changements apportés.
  • Naviguer dans l'historique de votre projet.

Accéder à la fiche de cours : Gestion des Versions avec Git

3. Créer un Repository sur GitHub et Pousser un Projet Local

Pour la collaboration à distance, GitHub est une plateforme incontournable. Cette fiche de cours vous guide pour créer un repository sur GitHub et y pousser vos projets locaux. Vous découvrirez :

  • Comment créer un repository sur GitHub.
  • Ajouter un remote à votre repository local.
  • Pousser vos fichiers locaux sur le repository distant.

Accéder à la fiche de cours : Pousser un Repository vers GitHub

Chacune de ces fiches est organisée de manière à vous offrir une expérience d'apprentissage claire, illustrée d'exemples pratiques et accompagnée de glossaires pour faciliter la compréhension des termes techniques. Ces ressources sont destinées aussi bien aux débutants qu'aux développeurs souhaitant se perfectionner dans l'utilisation de Git et GitHub.

Nous vous souhaitons une excellente exploration et beaucoup de succès dans votre Alchimie du Code !
