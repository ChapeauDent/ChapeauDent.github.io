JavaScript Learning Roadmap Checklist

Niveau Initial

Expliquer l’usage principal de JavaScript
Créer et utiliser une variable
Différencier let, const et var
Comprendre les types de données (string, number, boolean, undefined, null)
Utiliser typeof pour identifier le type d'une variable
Écrire un commentaire monoligne
Utiliser les opérateurs arithmétiques de base
Reconnaître, créer, affecter et lire un tableau
Reconnaître, créer, affecter et lire un objet
Afficher une variable dans la console (console.log, console.table)
Rechercher un élément dans le DOM (getElementById, querySelector)

Niveau Basique

Utiliser des opérateurs arithmétiques booléens
Créer un workflow conditionnel (if, else, switch)
Créer des boucles simples (for, while)
Créer sa propre fonction
Utiliser des fonctions fléchées (=>)
Appliquer des opérations simples aux chaînes de caractères
Appliquer des opérations simples aux tableaux
Réagir à des événements DOM de base (click, input)
Reconnaître et écrire des documents JSON
Utiliser console.table() pour afficher des tableaux/objets

Niveau Opérationnel

Utiliser des opérateurs d’affectation sur les variables (+=, -=, etc.)
Gérer des comparaisons de type égalité stricte (===)
Configurer des décisions conditionnelles complexes dans un workflow
Gérer les propriétés indéfinies
Ajouter un élément au DOM
Manipuler le DOM avec classList (ajouter/supprimer des classes)
Sérialiser et désérialiser JSON (JSON.stringify, JSON.parse)
Utiliser try...catch pour la gestion d’erreurs
Introduction à localStorage pour stocker des données

Niveau Avancé

Créer des boucles avancées (for...in, for...of)
Utiliser des variables spéciales (this, arguments)
Créer et affecter des objets complexes
Manipuler les propriétés d’un objet
Remplacer des éléments DOM
Fonctions anonymes et callbacks
Manipulation avancée des tableaux (map, filter, reduce)
Utilisation de setTimeout et setInterval

Niveau Expert

Maîtriser les opérations arithmétiques s’appliquant sur différents types
Manipuler des références d’objets
Réagir à des événements DOM avancés (event delegation, custom events)
Asynchrone avec async/await et fetch
Comprendre le this contextuel
Closures et portée lexicale
Modularisation avec import/export
